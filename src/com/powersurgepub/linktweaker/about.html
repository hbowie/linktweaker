LinkTweaker is written in Java. It may be run on Windows, Macintosh and other Unix platforms. LinkTweaker requires a Java Virtual Machine (JVM/JRE) of 1.5 (aka J2SE 5.0) or later.

LinkTweaker is distributed as a folder containing the file 'linktweaker.jar', along with other files. Keep the entire folder together. Double-click on 'linktweaker.jar' to launch the program.
